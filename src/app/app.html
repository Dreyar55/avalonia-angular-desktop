Avalonia + .NET 9 + Angular Desktop

Desktop app shell for the web — Angular inside Avalonia, powered by .NET

This project provides an ElectronJS / TauriJS-like experience but built on .NET. An Avalonia desktop app hosts a WebView and embeds an Angular app (can be any framework). The Angular frontend calls a local .NET API served by the Avalonia process and disposed on shutdown. Build desktop apps using familiar web development practices.
